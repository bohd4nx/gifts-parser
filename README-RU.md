## ✨ Возможности

- 🔍 **Эффективное сканирование** - Быстрая обработка тысяч участников чата
- ✨ **Поиск подарков** - Нахождение пользователей с лимитированными неапгрейжеными подарками (NFT)
- 📊 **Детальный вывод** - Получение ID пользователей, имён и информации о подарках
- 💾 **Экспорт результатов** - Сохранение результатов сканирования в текстовые файлы
- 🔄 **Пакетная обработка** - Обработка нескольких чатов за одну операцию
- 🌐 **Многоязычность** - Полная поддержка русского и английского интерфейсов
- 🔄 **Автообновление** - Автоматическое обновление кэша названий подарков каждый час

## 🛠 Установка

1. **Клонирование репозитория**:
   ```bash
   git clone https://github.com/bohd4nx/gifts-parser.git
   cd gifts-parser
   ```

2. **Установка зависимостей**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройка параметров**:
    - Отредактируйте `config.ini`:
        - Добавьте токен вашего Telegram бота (от [@BotFather](https://t.me/BotFather))
        - Укажите ваши Telegram API ID и API Hash (с [my.telegram.org](https://my.telegram.org))
        - Введите ваш номер телефона для аутентификации
        - Настройте ID администраторов и параметры парсинга

## 🚀 Использование

1. **Запуск бота**:
   ```bash
   python main.py
   ```

2. **Работа с ботом**:
    - Отправьте `/start` боту
    - Введите ссылку на Telegram-чат (или несколько ссылок на отдельных строках)
    - Дождитесь завершения сканирования
    - Скачайте результаты с помощью предоставленной кнопки

## 🔧 Пример использования и вывод

**Пример ввода**:

```
https://t.me/example_group
https://t.me/another_group
```

**Экран обработки**:

```
🔍 Парсинг чата [1/2]: @example_group
📊 Всего участников: 15K
👤 Обработано: 10K/15K (67%)
✨ Найдено пользователей: 42
⌛️ Осталось примерно: 5м
```

**Формат результатов**:

```
@username [12345678]
    Название подарка [ID_подарка]: 1 | Другой подарок [другой_ID]: 2
```

## 🌟 Продвинутое использование

- **Фильтр определенных подарков**: Отредактируйте `data/gifts.py`, раскомментировав карту подарков
- **Изменение размера пакета**: Настройте `BATCH_SIZE` в `config.ini` для более быстрой обработки (высокие значения
  могут вызвать ограничения скорости)
- **Несколько администраторов**: Добавьте разделенные запятыми ID пользователей в настройку `ADMINS`
